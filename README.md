# Retro Arcade Web - Backend

This is the Node.js / Express backend for the Retro Arcade Web mini-games platform.

## Features
- In-memory array mock database
- User authentication and session handling endpoints (`/api/login`, `/api/register`)

## Setup
1. Run `npm install` to install dependencies.
2. Run `node server.js` to begin the server on Port 3000.
3. Keep the terminal running so the frontend can interact with the authentication gates.
