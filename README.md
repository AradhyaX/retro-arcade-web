# Retro Arcade Web - Frontend

This is the frontend component of the Retro Arcade Web gaming platform, a single-page application (SPA) built using HTML, CSS, and basic JavaScript.

## Highlights
- Clean, responsive game dashboard grid layout
- Authentic casual gaming UI without excessive banners
- Permanent bottom navigation for quick access to various features (Home, Search, Account)
- Fully functional Tic Tac Toe mini-game integrated.

## Getting Started
To view this app locally:
1. Ensure the Node.js backend server (`../back-end/server.js`) is running.
2. Open `index.html` in an active browser window.
3. Access the `login`/`register` page directly from the UI or play games without an active session!
